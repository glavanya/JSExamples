JavaScript Array Methods

Shift()

The shift() method removes the first elementt of an array and shifts the other elements to the left

join()

The unshift() method adds new element to the beginning of an array and given the new length of an array

Length Property

The length Property provides an easy way to append new elements to an array without using the push() method.

Deleting elements leaves undefined holes in an array.

Concat()

The concat() method merges (concatenate arrays)
